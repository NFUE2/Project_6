# Pixel Adventure 

Fixel Adventure는 다른 플레이어들과 협동하는 멀티 2D 플랫포머RPG입니다.

![대체 텍스트](readme_images/start.png)

## ❓ 목표

직업을 선택 하고 앞으로 가면서 주위의 잡몹들을 해치우고 보스를 격파하는 것입니다.

![대체 텍스트](readme_images/play.png)

## 🕒 개발기간
2024.06.27 ~ 2024.08.22

## 🤝 팀원  
- 손효재
- 최도규
- 김성훈
- 함지운
  
## 🎮 조작
- 이동 : 방향키
- 점프 : SpaceBar
- 달리기 : Shift + 방향키
- 공격 : MouseLeft
- Q스킬버튼 : Q
- E스킬버튼 : E
